# Md. Shifat Bin Reza - Professional Portfolio

A modern, interactive professional portfolio showcasing expertise in **Software Quality Assurance (SQA)** and **Full-Stack Development**. Built with React 19, TypeScript, and Tailwind CSS, this portfolio highlights technical skills, work experience, and automation projects.

## 🛠️ Tech Stack

- **Frontend**: [React 19](https://react.dev/), [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/), [Framer Motion](https://www.framer.com/motion/) (Animations)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## ✨ Key Features

- **Interactive Navigation**: Seamlessly switch between sections (About Me, Experience, Projects, Skills, Education, Training).
- **Space-Themed UI**: A sleek, dark-themed interface with subtle starfield backgrounds and animations.
- **Dynamic Typewriter Effect**: Rotating QA roles to highlight multi-disciplinary expertise.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Timeline Components**: Structured presentation of work history and project highlights.
- **Data-Driven**: Content is decoupled from the UI, managed via a centralized `data.json` file.

## 📁 Project Structure

```text
src/
├── components/      # Reusable UI components (SkillTag, TimelineItem, Typewriter)
├── assets/          # Static assets (Images, SVGs)
├── App.tsx          # Main application logic and routing
├── data.json        # Portfolio content (Personal info, experience, projects)
├── types.ts         # TypeScript interfaces for data structure
└── index.css        # Global styles and custom "Space Theme"
```

## ⚙️ Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shifat97/new-portfolio-react.git
   ```
2. Navigate to the project directory:
   ```bash
   cd new-portfolio-react
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Build the production-ready application:
```bash
npm run build
```
